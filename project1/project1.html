SEO BEST PRACTICES, ACCESSIBILITY, USER ENGAGEMENT

How do search engines work?

Search engines allow users to enter queries then the system returns relevant results based on the query. Search engines use ranking factors in order to determine how relevant and popular a site is. To parse through the various websites on the internet, search engines also use keywords in order to determine relevancy to the query. Some companies even pay to have their sites appear first or on the first page of a relevant query (Lab, n.d.).

The importance of keywords

Using keywords and descriptions as meta tags is important when creating websites so that when users make a query in a search engine, the website can appear. Making keywords specific, it narrows the competition in the search results and improve the websites ranking in the search list. The meta description tag is also vital in helping a website get some traffic. The meta description tag allows users to read a small description or snippet of the page on the search engine. Thar snippet or description would be the indicator of which website would ultimately be chosen by the searcher. Therefore, it is imperative that the description is compelling (MOZ, n.d.).

Four elements that comprise writing relevant content links

There are four elements that comprise of writing relevant content links and they are experience, expertise, authoritativeness, and trustworthiness, also known as, EEAT (Muller, 2019). Websites that do not adhere to EEAT will not be able to gain visibility on search engines. Websites must show that the author has experience on the topic of their website. If it is obvious that the author has no experience on the topic, they lose credibility. The author must also show they have expertise in the subject, this can be done through showing research, credentials, or linking to a site that is a trusted authority. Trusted authorities, or Authority, are site that are inherently considered trustworthy like government sites. Lastly, the site has to establish that it is trustworthy. This can be done by earning links from other trustworthy sites (Moz, 2025).

Good heading hiearchy with example

Headings are uses to show document structure. The headings should be organized in order of importance. The h1 heading should be the most important which is typically the page title. Then the h2 tag should be used for secondary headings like section titles. The h3 heading would be used for things like sub-sections (w3schools, 2019). Below is an example of what this could look like:

h1 Sunscreen is Vital to Skin Care /h1

p Your dermatologist was not nagging when they said that sunscreen should be a part of your daily routine /p

h2 Sun Exposure is making you look old! /h2

and once it is on the public or local site it will look like the following:

Sunscreen is Vital to Skin Care

Your dermatologist was not nagging when they said that sunscreen should be a part of your daily routine

Sun Exposure is making you look old!
